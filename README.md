# Pendidikan Berkualitas - School Website

A web-based school information system developed using PHP and HTML, with a modern and responsive user interface styled using Tailwind CSS (CDN). This project runs on a local server environment using XAMPP.

This website provides information about education, articles, and allows users to explore educational content through a clean and user-friendly interface.

---

## Tech Stack

- PHP
- HTML
- Tailwind CSS (CDN)
- MySQL
- XAMPP (Local Server)

---

##  Features

- Homepage with hero section
- Article page to display educational content
- About Us page
- Contact page
- Comment section
- Dark mode toggle
- Responsive design (mobile-friendly)
- Modern UI using Tailwind CSS

---
